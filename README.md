# Zigshot

Screenshot tool written in zig for _wayland compositors_ implementing **zwlr_screencopy_unstable_v1 protocol.**

# Dependencies:

1. `zig` 0.9.1
1. `wayland`
1. `wayland-protocols`

# Building

```console
$ git clone https://github.com/shinyzenith/zigshot;cd zigshot
# zig build --prefix /usr/local
```
